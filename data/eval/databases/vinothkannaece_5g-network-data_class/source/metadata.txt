This project simulates a large-scale 5G network performance dataset using Python and pandas in Google Colab. It generates synthetic data for various real-world metrics like signal strength, speed, latency, jitter, battery levels, and user experience across different cities — with special focus on Indian metro areas like Mumbai, Chennai, Delhi, and Kolkata. The dataset includes detailed attributes like device model, carrier (Jio, Airtel, Vi), 5G frequency bands (n78, n28, n258), and environmental conditions. This data can be used for exploratory data analysis (EDA), network optimization modeling, predictive analytics, and machine learning experiments related to telecom service quality.
