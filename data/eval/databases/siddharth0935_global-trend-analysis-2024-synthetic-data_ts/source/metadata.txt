This synthetic dataset mimics 2024 trends across technology, finance, and sustainability. It includes:  
- **10,000 records** with timestamps (Jan 2023 - Dec 2024).  
- **Key metrics**: Weekly popularity, sentiment scores, growth rates, and regional data.  
- **Use Cases**: Time-series forecasting, sentiment analysis, and trend clustering.  

*Disclaimer: Data is entirely synthetic and not based on real-world observations.*  

=== About this file ===

About this file
Update Frequency: Static (one-time simulation)
Dataset Size: 10,000 rows (~1.5 MB)
Generated With: Python (Faker, Pandas, NumPy)


=== Columns & descriptions ===

trend_id: Unique identifier for each trend.
trend_name: Name of the trend (e.g., "AI Agents").
category: Trend category (Tech/Finance/Sustainability).
date: Date of record (YYYY-MM-DD).
weekly_interest: Normalized popularity score (0-100).
region: Geographic origin of the trend.
sentiment: Social media sentiment (-1 to +1)
related_brands: top brands associated with the trend
growth_rate: Month-over-month change in interest (%).
forecast_2025: Predicted interest score for 2025 (0-100).
