# Context
The 1985 Auto Imports Dataset captures a pivotal era in automotive history, documenting the shift toward fuel efficiency and globalization in car manufacturing. This dataset enables:
* Analysis of 1980s automotive trends (MPG, engine tech, pricing)
* Predictive modeling for insurance risk (symboling) and resale value

# Overview
| Category        | Detail                          |
|-----------------|---------------------------------|
| Records         | 205 records     |
| Timeframe       | 1985        |
| Manufacturers         | 22 brands (e.g., Toyota, BMW, Chevrolet)|
| Features      | Price, MPG, horsepower, body style, fuel type |

# Files
Main Dataset (data.csv)
* Standardized missing values (? → NaN)
* Corrected dtypes (e.g., horsepower as float)
* Original columns preserved with improved readability

# Key Features
1. Reliable dtypes: No more ValueError issues (fixed NaN and numeric columns).
2. Historical Snapshot: Documents 1985 market trends (pre-NAFTA, pre-electronic engines).
3. Machine Learning Ready: Cleaned for regression (predict price) or classification (e.g., fuel-type).
4. Geographic Diversity: Mix of US, European, and Japanese manufacturers.

# Uses
This dataset supports:
* Price Prediction: Train models using engine size, MPG, and brand.
* Risk Analysis: Correlate symboling with safety features.
* Fuel Efficiency Studies: Compare 1985 MPG standards to modern EVs.
* EDA Tutorials: Ideal for teaching pandas/seaborn (small but feature-rich).

=== Columns & descriptions ===

symboling: Index
normalized-losses: Insurance risk rating (-3 to +3), where +3 = high risk
make: Average insurance claim payout per vehicle (relative to other cars)
fuel-type: Manufacturer (e.g., "toyota", "bmw")
aspiration: "gas" or "diesel"
num-of-doors: Engine air intake ("std" or "turbo")
body-style: "two" or "four"
drive-wheels: "sedan", "hatchback", "wagon", etc.
engine-location: "4wd", "fwd" (front-wheel), "rwd" (rear-wheel)
wheel-base: "front" or "rear"
length: Distance between wheels (inches)
width: Vehicle dimensions (inches)
height: Vehicle dimensions (inches)
curb-weight: Vehicle dimensions (inches)
engine-type: Weight without passengers (lbs)
num-of-cylinders: Engine design (e.g., "ohc", "ohcv")
engine-size: "two", "four", "six", etc.
fuel-system: Displacement volume (cubic inches)
bore: Fuel delivery (e.g., "mpfi", "2bbl")
stroke: Engine cylinder dimensions (inches)
compression-ratio: Engine cylinder dimensions (inches)
horsepower: Ratio of cylinder volume (high = more efficient)
peak-rpm: Engine power (HP)
city-mpg: Engine speed at max power (RPM)
highway-mpg: Fuel efficiency (miles per gallon)
price: Fuel efficiency (miles per gallon)
