#### **Overview**  
This dataset provides **historical daily stock price data** for the **20 largest financial companies** and **major global stock indices**. It is designed for financial analysis, market trend exploration, and correlation studies. The dataset includes stock price history for the past **1 year** for financial companies and for stock indices.  

#### **Contents**  
The dataset consists of multiple CSV files:  
1. **Stock data files (`stock_JPM.csv`, `stock_BAC.csv`, etc.)** ‚Äì Each file contains daily price data for a specific financial company.  
2. **`indices.csv`** ‚Äì A merged dataset containing daily closing prices for major stock indices.  

---

### **Dataset Structure**  

#### **1. Stock Data Files (`stock_JPM.csv`, `stock_BAC.csv`, etc.)**  
Each stock file contains the following columns:  
- **`Date`** ‚Äì Trading day‚Äôs closing date (`YYYY-MM-DD` format).  
- **`Open`** ‚Äì Opening price of the stock.  
- **`High`** ‚Äì Highest price during the trading day.  
- **`Low`** ‚Äì Lowest price during the trading day.  
- **`Close`** ‚Äì Closing price of the stock.  
- **`Volume`** ‚Äì Number of shares traded.  

**Removed Columns:**  
- **`Dividends`** ‚Äì Removed to focus on price movements.  
- **`Stock Splits`** ‚Äì Removed as it does not impact short-term trading analysis.  

Each file represents a **single company‚Äôs** daily stock performance over the past **365 days**.  

---

#### **2. Stock Indices File (`indices.csv`)**  
This file contains historical **daily closing prices** for major global stock indices over the past **six months**.  

**Columns:**  
- **`Date`** ‚Äì Trading day‚Äôs closing date (`YYYY-MM-DD` format).  
- **`GSPC`** ‚Äì **S&P 500 Index**.  
- **`IXIC`** ‚Äì **NASDAQ Composite Index**.  
- **`DJI`** ‚Äì **Dow Jones Industrial Average (DJIA)**.  
- **`RUT`** ‚Äì **Russell 2000 Index**.  
- **`GDAXI`** ‚Äì **DAX Index (Germany)**.  
- **`FTSE`** ‚Äì **FTSE 100 Index (United Kingdom)**.  
- **`FCHI`** ‚Äì **CAC 40 Index (France)**.  
- **`HSI`** ‚Äì **Hang Seng Index (Hong Kong)**.  
- **`N225`** ‚Äì **Nikkei 225 Index (Japan)**.  
- **`BSESN`** ‚Äì **BSE Sensex Index (India)**.  
- **`BVSP`** ‚Äì **Bovespa Index (Brazil)**.  

**Key Adjustments:**  
- The **ticker for `S&P 500` (`^GSPC`) has been renamed to `GSPC`** to avoid special characters.  
- The file **does not include separate index files**, only the merged dataset.  
- Missing values are **forward-filled** to maintain continuity in time series analysis.  

---

### **Data Sources & Processing**  
- Stock and index data were **collected and processed** using the **Yahoo Finance API**.  
- The dataset was prepared with **[equity_analysis](https://github.com/niki8885/equity_analysis/tree/master/equity_analysis)**, a custom module developed for efficient equity data retrieval.  
- The data updates cover:  
  - **Financial companies:** Last **365 days** of daily price data.  
  - **Stock indices:** Last **180 days** of daily closing prices.

=== About this file ===

About this file
The indices.csv file contains historical daily closing prices for major global stock indices.
If a market was closed on a particular day (e.g., holidays), the corresponding values might be missing.
Missing values are forward-filled using the last available price.
All dates are provided in UTC without timezone adjustments.
