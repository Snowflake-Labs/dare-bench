# 🧾 Financial Transactions 

This inn dataset provides a small but representative sample of anonymized financial transactions for building and testing fraud detection models.

Each record corresponds to a single transaction and includes detailed information such as transaction type, amount, sender and receiver account balances, and fraud indicators. It is suitable for binary classification, anomaly detection, and other machine learning tasks related to financial security.

## ⚠️ Class Imbalance Notice

&gt; **Important:**  
&gt; This dataset is **highly imbalanced** — the number of fraudulent transactions (`isFraud = 1`) is much lower compared to non-fraudulent ones.  
&gt; This reflects real-world financial data and may affect model performance if not handled properly.

To improve results, consider:
- **Resampling techniques** like SMOTE or undersampling
- Using **evaluation metrics** like precision, recall, F1-score, or ROC-AUC instead of just accuracy
