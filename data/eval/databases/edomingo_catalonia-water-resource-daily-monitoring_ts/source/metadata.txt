This **Dataset** contains daily readings from **310 water sensors** (Reservoir Level Sensors, River Gauge Sensors, Pluviometer Sensors, and Aquifer Piezometer Sensors) placed around **Catalonia** (Spain). Everyday the readings from the previous day at 0:00h, 6:00h, 12:00h and 18:00h are collected and aggregated to each table with an automatically [Scheduled 🔧🗃️ETL Kaggle Notebook](https://www.kaggle.com/code/edomingo/etl-catalonia-water-resource-monitoring/notebook)   in order to extract this information from the public API of the [ACA - Agència Catalana de l'Aigua](https://aca.gencat.cat/) (Catalan Water Agency). This API serves the real-time value reads from the sensors, but it also contains detailed Metadata information for each sensor which has been extracted and saved from another supplementary [💾Notebook](https://www.kaggle.com/code/edomingo/metadata-cat-water-resource-monitoring) , and imported here into the `metadata/` folder.

## Dataset Motivation and Context:

In Catalonia, like in other Mediterranean regions, we are currently experiencing a **dry period**, characterized by **reduced precipitation and lower-than-average water levels** in reservoirs and rivers. It's not summer yet and we are already struggling in our water reserve levels, for this reason the local authorities are starting to plan a set of actions (e.g. water supply reductions/cuts) in the following months. Gathering this data, starting from early January and continuosly updating it day by day, will be a great source for further real-time Analysis of the situation, Visualizations of trends, and even the possibility to build short term Prediction Models. 🚰📉

CSV file where every row is the sensor read value at 00:00h, 06:00h, 12:00h, 18:00h for each day for everyone of the gauge sensors whose ids are in the column names. If the sensor is Out of Service, failing or there are no reads from it for the given time range, the cell will contain a Nan. You can find more info about the description, units, ranges, state, location, and more in the metadata file of the gauge sensors:
